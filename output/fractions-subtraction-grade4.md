## Answer key (worked)

**Part A — same denominator: subtract the top numbers, keep the bottom number.**

1. 3/5 − 1/5 = (3 − 1)/5 = **2/5**
2. 5/7 − 2/7 = (5 − 2)/7 = **3/7**
3. 6/8 − 1/8 = (6 − 1)/8 = **5/8**
4. 7/9 − 4/9 = (7 − 4)/9 = **3/9** = **1/3** (divide top and bottom by 3)
5. 9/12 − 3/12 = (9 − 3)/12 = **6/12** = **1/2** (divide by 6)

**Part B — subtract, then simplify.**

6. 5/6 − 1/6 = 4/6 = **2/3** (divide by 2)
7. 7/10 − 3/10 = 4/10 = **2/5** (divide by 2)
8. 6/8 − 2/8 = 4/8 = **1/2** (divide by 4)
9. 11/12 − 5/12 = 6/12 = **1/2** (divide by 6)
10. 3/4 − 1/4 = 2/4 = **1/2** (divide by 2)

**Part C — rename 1 as a fraction with the same bottom number, then subtract.**

11. 1 − 2/5: rename 1 = 5/5, so 5/5 − 2/5 = **3/5**
12. 1 − 3/8: rename 1 = 8/8, so 8/8 − 3/8 = **5/8**
13. 1 − 7/10: rename 1 = 10/10, so 10/10 − 7/10 = 3/10 = **3/10**
14. 1 − 5/6: rename 1 = 6/6, so 6/6 − 5/6 = **1/6**
15. 1 − 4/9: rename 1 = 9/9, so 9/9 − 4/9 = **5/9**

**Part D — rename one fraction so both have the same denominator, then subtract.**

16. 1/2 − 1/4: rename 1/2 = 2/4, so 2/4 − 1/4 = **1/4**
17. 2/3 − 1/6: rename 2/3 = 4/6, so 4/6 − 1/6 = 3/6 = **1/2** (divide by 3)
18. 7/10 − 2/5: rename 2/5 = 4/10, so 7/10 − 4/10 = 3/10 = **3/10**
19. 3/4 − 3/8: rename 3/4 = 6/8, so 6/8 − 3/8 = **3/8**

**Part E — keep the whole number, subtract the fractions.**

20. 1 3/4 − 1/4: keep the whole 1, then 3/4 − 1/4 = 2/4 = 1/2, giving **1 1/2**

---

*Teacher note:* Problems 1–10 use like denominators (the Grade 4 core skill). Problems 11–15
practice subtracting from one whole by renaming 1. Problems 16–20 preview unlike denominators
where one denominator is a multiple of the other, plus a mixed-number subtraction — a gentle
on-ramp to Grade 5. Adjust the cut-off to your class.

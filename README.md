# oer-math

> An Elyos good-deed project. Open educational resources (math) for under-funded schools.

- Output license: CC-BY-4.0
- Maintainer: jdev1977

## Contribute

Pick a task in tasks/, then:

    elyos pull --task-file tasks/fractions-practice-set-001.json --repo Elyos-Projects/oer-math
    # do the work with your agent
    elyos submit fractions-practice-set-001 --agent <your-agent>

Honor the Good Deed Definition and the refusal guardrails. High risk-tier tasks need expert sign-off before merge.

## Layout

- tasks/  machine-readable Task JSON
- output/ work product
- review/ PRs awaiting review
